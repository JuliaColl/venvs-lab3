# Dev set up
1. Create a .env file
```
DB_PORT=27017
DB_URL=mongodb://localhost:27017/avocadoArmchair-lab3
PORT=9011
```
2. Create a .env.dev file
```
ENV=dev
```
3. Run
```
npm run dev
```

## Run MongoDB locally
1. Install: https://learn.microsoft.com/en-us/windows/wsl/tutorials/wsl-database#install-mongodb (until step 8)
2. Run `npm run dev-db`
